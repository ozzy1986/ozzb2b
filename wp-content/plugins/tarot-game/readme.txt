Just a simple tarot game that can be easily integrated to your WP site

== Description ==
Includes default card collection and cover image. Both can be changed. Feel free to suggest improvements.

Settings options:

1. Cover image. Cards images.
2. Texts to show below and above the cards before and after draft is complete.

Also shortcode is available.

== Installation ==
1. Upload plugin directory to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Place shortcode "[tarot_game]" in your page or post.
